# homebrew-monokle

## Prerequisite

You should tap into our repository as follows:

```bash
brew tap kubeshop/monokle
```

## Monokle Desktop

You can install Monokle Desktop as a cask:

```bash
brew install monokle
```

## Monokle CLI

You can install Monokle CLI as a formula:

```bash
brew install monokle-cli
```
